CABLE CREATION LOGIC:
def create_slack_cable(start, end, slack, use_gravity=False):
    # Create curve data
    curve = bpy.data.curves.new('Cable', 'CURVE')
    curve.dimensions = '3D'
    
    # Create spline
    spline = curve.splines.new('BEZIER')
    spline.bezier_points.add(2)  # 3 points total
    
    # Set points
    points = spline.bezier_points
    midpoint = (start + end) / 2
    
    if use_gravity:
        # Calculate catenary sag based on distance
        distance = (end - start).length
        sag = slack * (distance / 2)
    else:
        sag = slack
        
    midpoint.z -= sag  # Drop the middle
    
    points[0].co = start
    points[1].co = midpoint
    points[2].co = end
    
    # Set handles
    for p in points:
        p.handle_type_left = 'AUTO'
        p.handle_type_right = 'AUTO'
    
    # Bevel for thickness
    curve.bevel_depth = thickness
    curve.bevel_resolution = 4
    
    return curve

ERROR HANDLING:
- Check exactly 2 objects selected for object mode
- Check exactly 2 verts selected for edit mode
- Undo support
- Report cable length on creation
