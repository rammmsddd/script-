# Slack Cable Generator

A Blender addon for creating realistic cables, pipes, and electrical conduits with catenary curves and professional materials.

![Blender](https://img.shields.io/badge/Blender-3.0%2B-orange?logo=blender)
![Version](https://img.shields.io/badge/Version-1.1.0-blue)
![License](https://img.shields.io/badge/License-GPL--3.0-green)

## Features

- **Cable Between Objects** — Create slack cables between two selected objects with adjustable sag
- **Chain Cable** — Route cables through multiple points in sequence
- **Cable Bundle** — Generate multiple cables with natural variation
- **Pipe Route** — Create conduits with smooth 90° bends
- **Material Presets** — Black, Red, Yellow, Blue, Green, White, Copper, and Steel

## Installation

1. Download `__init__.py`
2. Open Blender → Edit → Preferences → Add-ons
3. Click "Install" and select the file
4. Enable "Slack Cable Generator"

## Usage

1. Open the **SaaS Tools** panel in the 3D Viewport sidebar (N key)
2. Select two objects as start/end points
3. Adjust slack, thickness, and resolution
4. Click the desired cable type button

## Parameters

| Parameter | Description |
|-----------|-------------|
| Slack | Cable sag amount |
| Thickness | Cable/pipe diameter |
| Resolution | Curve smoothness |
| Bend Radius | Corner radius for conduits |

## Compatibility

- Blender 3.x, 4.x, 5.x
